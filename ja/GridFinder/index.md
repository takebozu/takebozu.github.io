# <img src="/img/icons/gridfinder_logo.png" height="32" align="top" alt="GridFinderアイコン"> GridFinder (iOS)

GridFinder は「どこを探し終えたか」がひと目で分かるアプリです。

歩いた場所が地図の上にメッシュ（格子）となって塗られていきます。通ったマスが埋まっていくので、まだ探していない場所と探し終えた場所が一目瞭然。同じ場所を二度探したり、隅を見落としたりすることがなくなります。

「くまなく探す」あらゆる場面に:

- 落とし物・鍵・持ち物の捜索
- 迷子になったペット探し
- 山菜・きのこ狩り、ビーチコーミング
- 地上での捜索、フィールド調査、見回り

## 主な機能

- **塗りつぶしカバレッジ** — 通った経路がマス単位で地図に記録され、探し終えた範囲が埋まっていきます
- **解像度の切り替え** — 5m / 10m / 20m から選べ、記録の細かさを調整できます
- **自動停止タイマー** — 指定した時刻で自動的に記録を止められます
- **探索履歴** — セッションは自動保存。過去の探索を開いて地図に重ね、どこを歩いたか比較できます
- **みんなで協力探索**（任意） — 8文字のコードを共有すれば、仲間と一緒に広い範囲を分担探索。お互いの探索済みマスがリアルタイムで見えるので重複しません。自分のマスは濃い色、仲間のマスは薄い色で表示されます

GridFinder は地図と現在地だけに集中した、シンプルで実用的なツールです。

※ リアルタイム協力探索はチケット（アプリ内課金）を使用します。1人での探索と履歴は常に無料です。
